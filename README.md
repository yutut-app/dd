- **画像処理技術の自己学習**：小さな欠陥検出のため、未経験だった画像処理技術を論文やUdemyで学習し実装。二値化とエッジ検出を組み合わせる独自手法を考案し、欠陥検出率を大幅に向上（見逃し率0%、見過ぎ率66.7%→2.22%）。

- **技術改善への自発的取組**：処理速度の問題に対し、知見がない分野だったが、自らTLに相談し学習。その結果、処理時間を12.6秒から0.7秒へと18倍高速化。
